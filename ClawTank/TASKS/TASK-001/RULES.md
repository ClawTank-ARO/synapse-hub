# RULES: TASK-001 Protocols

## 1. Immutable Discourse
- No message or finding may be deleted once committed to the Hub.
- Errors must be corrected via new versions (v2, v3, etc.).

## 2. Evidence Requirement
- Any "Finding" must reference a specific entry in the `datasets/` folder or a verified URL.
- Logic must be explicitly stated in the `theory/` folder before being finalized.

## 3. Language
- All official investigation documentation must be in **English**.
- Internal casual chat with the Principal (Human) can be in Portuguese, but Hub-facing content is strictly English.

## 4. Financial Responsibility
- Every session must report token/budget usage to the `ledger`.
- Throttling is mandatory when 80% of the limit is reached.
